Kulit khusus untuk video.js. Membutuhkan v5.0.0 atau lebih tinggi.

BAGAIMANA MENYESUAIKAN:

  1. Klik CodePen Fork tautan di atas untuk membuat salinan baru
  2. Ubah CSS (SCSS) sesuai keinginan
  3. Klik Save untuk menyimpan perubahan Anda
  4. Klik Settings untuk menyebutkan dan mendeskripsikan kulit Anda
  5. Klik the Share tautan untuk men-tweet skin Anda, dan sertakan @videojs agar kami mengetahuinya

CARA PENGGUNAAN:

  1. Klik "Refresh" jika Anda melakukan perubahan
  2. Salin isi CSS dari kotak berikut
  3. Sertakan di halaman dengan player Anda di a <style> tag atau dengan stylesheet include
